起源 // 人生路

选择你的人生路

你的过去定义了你在夜之城的开局。三条截然不同的人生路，三段独特的故事起点。

🏜

流浪者

来自恶土荒野的自由灵魂。你离开了家族部落，独自驶向夜之城的霓虹灯海。你了解荒野的法则，但城市的规矩对你来说是全新的。

🏙

街头小子

你在夜之城的街头长大，混凝土就是你的操场。你了解每一条暗巷、每一个帮派头目，以及在这座城市生存的潜规则。

🏢

公司人

荒坂公司的精英特工。你曾在权力金字塔的顶端俯瞰众生，直到被自己的上司出卖，一夜之间跌落谷底。

档案 // 关键人物

你在夜之城的战友

在这座每个人都可能背叛你的城市里，这些人选择了站在你身边。

👤

强尼·银手

摇滚小子 / 数字幽灵

传奇摇滚歌手，反企业斗士。他的意识被存储在Relic芯片中，如今寄生在V的大脑里。他想要的只有一件事——毁灭荒坂。

💀

杰克·威尔斯

V的搭档 / 海木之子

海木区长大的硬汉，V最忠诚的搭档。他梦想着出人头地，成为夜之城的传奇。他是你在这座城市里最值得信赖的人。

🔧

朱迪·阿尔瓦雷兹

超梦技师 / 反叛者

才华横溢的超梦编辑师，对夜之城的腐朽深恶痛绝。她想要改变世界，但这座城市似乎不允许任何人保持纯真。

⚡

帕南·帕尔默

流浪者 / 奥德卡恩部落

脾气火爆的流浪者佣兵。她脱离了原来的部落，独自闯荡恶土。凭借出色的驾驶技术和不屈的意志，她是荒野中最可靠的战友。
